3.3 方法
方法的定义包括两部分：方法声明和方法体
一般格式为：
```
    方法声明部分
    {
        方法体内容
    }
```

1. 方法声明部分（方法头）
由方法的类型，名称和一对小括号以及其中的参数列表所构成
例如：
```
    int speak() //无参数的方法头
    {
        return 23;
    }
    int add (int x,int y,int z) //有参数的方法头
    {
        return x+y+z;
    }
```
- 方法返回的数据类型可以任意，**当一个方法不需要返回数据时，返回类型必须是void**。
- 可以有多个return

---

2. 方法体
方法体+局部变量的声明+Java语句。
例如
```
    int getAvg(int n){  //参数变量n是局部变量，n在整个方法内都有效
        int avg;
        int sum=0;  //声明局部变量sum，作用域是本行及下方5行
        for(int i=1;i<=n;i++){  //i作用域在for循环语句内
            sum=sum+i;  //sum在第一次被使用之前必须被赋值
        }
        avg=sum/n;
        return avg; //return语句
    }
```

3.4 成员变量和局部变量
1. 有效性（单选）
局部变量只在方法内有效，与其声明的位置有关。

方法的参数在整个方法内有效，方法内的局部变量从声明它的位置之后开始有效。

**复合语句**中生命的局部变量的有效范围是该复合语句

**循环语句**中声明的局部变量的有效范围**是该循环语句**。

例如：
```
    public class A{
        void f(){
            int m=10,sum=0; //成员变量，在整个类中有效
        }
    }
```

2. 默认值（单选）
成员变量有默认值，局部变量没有默认值，在使用局部变量之前，必须保证局部变量有具体的值。
例如：
下列Point类无法通过编译
```
    class Point{
        int x=10,y;
        void getX(){
            int m;
            x=y+m;
        }
    }
```

3. 成员变量的隐藏（阅读程序）
若**局部变量的名与成员变量的名相同**，则成员变量被隐藏，即**该成员变量在这个方法内暂时失效**
```
    class Point{
        int x=10,y;
        void getY(){
            int x=5;
            y=x+x;
        }
    }
```
